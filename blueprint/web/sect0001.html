  • LinearAlgebraGame.VectorSpace

A vector space is a space over a field K with an abelian group V. It has four main properties:

  • smul_add: \(\forall (a : K) (x y : V), a \bullet (x + y) = a \bullet x + a \bullet y \)

  • add_smul: \(\forall (a b : K) (x : V), (a + b) \bullet x = a \bullet x + b \bullet x \)

  • mul_smul: \(\forall (a b : K) (x : V), (a * b) \bullet x = a \bullet (b \bullet x) \)

  • one_smul: \(\forall (x : V), (1 : K) \bullet x = x \)
